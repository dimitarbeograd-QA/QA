# Contributing

## Как да допринесеш
1. `npm install`
2. Направи промените си (маршрути, тестове).
3. `npm test` — увери се, че целият suite минава преди commit.
4. Нова функционалност трябва да идва с придружаващ Mocha/Chai тест.
5. Копирай `sample.env` в `.env` локално (никога не commit-вай `.env`).
